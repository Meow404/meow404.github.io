Maze-Traversal Bot

Arduino Embedded Systems Path Planning Sensor Fusion

An 8-person multi-disciplinary project where the objective was to build a robot that could autonomously traverse and map an unknown maze. The team spanned hardware, software, and algorithms — requiring tight cross-functional coordination over a single semester.

I led the hardware team, responsible for designing and building the Arduino-based physical platform:

  • Sensor suite: ultrasonic and infrared sensors for wall detection and distance measurement, calibrated for consistent readings across maze materials.
  • Motor control: a dual-motor drive system with PID-controlled speed, enabling precise turns and straight-line tracking.
  • Command interface: the hardware layer exposes a clean serial protocol, letting the algorithm team send high-level movement commands (turn left, go forward N cm) without worrying about motor tuning.

The final system successfully traversed and fully mapped the competition maze, placing competitively against other teams.
